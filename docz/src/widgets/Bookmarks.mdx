## `widgetArgs` Prop
The `widgetArgs` prop accepts properties of the widget as defined in the official docs. See [https://developers.arcgis.com/javascript/latest/api-reference/esri-widgets-Bookmarks.html](https://developers.arcgis.com/javascript/latest/api-reference/esri-widgets-Bookmarks.html) for the full list of properties available.
<Playground>
    <div style={{height: "500px", width: "100%"}}>
        <Map id="widgetArgsBookmarks" webMapId="f2e9b762544945f390ca4ac3671cfa72">
            <Bookmarks widgetArgs={{
                editingEnabled: true,
                bookmarkCreationOptions: {
                    takeScreenshot: true,
                    captureViewpoint: false,
                    screenshotSettings: {
                        width: 100,
                        height: 100
                    }
                }
            }}
            />
        </Map>
    </div>
</Playground>


## `expander` Prop
The `expander` contains the layer list behind an icon button that can toggle.


<Playground>
    <div style={{height: "500px", width: "100%"}}>
        <Map id="BookmarksExpander" webMapId="f2e9b762544945f390ca4ac3671cfa72">
            <Bookmarks expander />
        </Map>
    </div>
</Playground>

## `expandIconClass` Prop
The `expandIconClass` prop allows you to change the icon of the expander. you can choose other classes from here [https://developers.arcgis.com/javascript/latest/esri-icon-font/](https://developers.arcgis.com/javascript/latest/esri-icon-font/)


<Playground>
    <div style={{height: "500px", width: "100%"}}>
        <Map id="BookmarksExpanderIcon" webMapId="f2e9b762544945f390ca4ac3671cfa72">
            <Bookmarks expander expandIconClass={'esri-icon-collection'} />
        </Map>
    </div>
</Playground>

## `expandWidgetArgs` Prop
The `expandWidgetArgs` prop allows you to add any additional arguments to the expand constructor. see the official docs for what you can add here.
[https://developers.arcgis.com/javascript/latest/api-reference/esri-widgets-Expand.html](https://developers.arcgis.com/javascript/latest/api-reference/esri-widgets-Expand.html)


<Playground>
    <div style={{height: "500px", width: "100%"}}>
        <Map id="BookmarksExpandWidgetArgs" webMapId="f2e9b762544945f390ca4ac3671cfa72">
            <Bookmarks 
                expander
                expandIconClass={'esri-icon-collection'}
                expandWidgetArgs={{
                    mode: "drawer",
                    expandTooltip: "Custom Tooltip"
                }}
            />
        </Map>
    </div>
</Playground>
